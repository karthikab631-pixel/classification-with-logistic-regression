# Logistic Regression Classification

## Objective
Build a binary classification model using Logistic Regression.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Breast Cancer Dataset (Scikit-learn)

## Steps
1. Load dataset.
2. Split into training and testing sets.
3. Standardize features.
4. Train Logistic Regression model.
5. Predict test data.
6. Evaluate using:
   - Confusion Matrix
   - Precision
   - Recall
   - ROC-AUC Score
7. Tune the decision threshold.
8. Explain the sigmoid function.

## Results
The Logistic Regression model achieved high accuracy with excellent precision, recall, and ROC-AUC, making it effective for binary classification.
